# <runningorder>

## サイト概要
管理者が主催する大会に会員のランナーが参加して順位を競うことができる

### サイトテーマ
大会に参加する、ランナーの記録がわかる

### テーマを選んだ理由
ランナーの目標が作れるようにするため

### ターゲットユーザ
地元ランナー、サークルなど

### 主な利用シーン
大会や記録会に気軽に参加できる
